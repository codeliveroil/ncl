ncl - Navigable Calendar
========================

A navigable version of the in-built calendar (cal).

Works on: macOS, Linux

Installation
------------
Download the latest release from the releases page and copy it to /usr/local/bin

Usage
-----
```
ncl -h
```

Examples
--------
```
ncl
ncl 8
ncl 8 2017
```
